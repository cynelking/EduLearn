Data Science — Python

Un cours complet pour apprendre les bases et techniques essentielles de la Data Science en Python.

Introduction à la Data Science

La Data Science combine les statistiques, la programmation et la visualisation pour analyser et interpréter les données. Python est aujourd’hui le langage le plus populaire grâce à sa simplicité et ses puissantes bibliothèques.

Chapitre 1 : Les bases de Python

Python est très utilisé grâce à sa syntaxe simple :

# Afficher un texte
print("Bonjour Data Science !")

# Créer une variable
x = 10

# Une liste
data = [1, 5, 7, 9]

# Une boucle
for i in data:
    print(i)

Vous devez maîtriser les variables, listes, dictionnaires, boucles et fonctions pour continuer.

Chapitre 2 : Manipulation des données avec Pandas

Pandas est la bibliothèque principale pour analyser des tableaux de données.

import pandas as pd

# Charger un fichier CSV
df = pd.read_csv("data.csv")

# Afficher les 5 premières lignes
print(df.head())

# Informations générales
print(df.info())

# Sélection d'une colonne
ages = df["Age"]

# Moyenne d'une colonne
print(df["Salaire"].mean())

Avec Pandas, vous pouvez nettoyer, trier, transformer et analyser vos datasets.

Chapitre 3 : Analyse statistique

Python propose plusieurs outils statistiques :

import numpy as np

data = [10, 12, 9, 14, 18]

print("Moyenne :", np.mean(data))
print("Médiane :", np.median(data))
print("Écart-Type :", np.std(data))

Chapitre 4 : Visualisation des données

Matplotlib permet de faire des graphiques :

import matplotlib.pyplot as plt

x = [1, 2, 3, 4]
y = [10, 20, 25, 30]

plt.plot(x, y)
plt.title("Évolution des valeurs")
plt.xlabel("Temps")
plt.ylabel("Valeurs")
plt.show()

Vous pouvez aussi utiliser Seaborn pour des graphiques plus stylisés.

Chapitre 5 : Introduction au Machine Learning

Pour créer un modèle prédictif :

from sklearn.linear_model import LinearRegression
import numpy as np

# Données d'entraînement
X = np.array([[1], [2], [3], [4]])
y = np.array([3, 6, 9, 12])

# Créer le modèle
model = LinearRegression()

# Entraîner
model.fit(X, y)

# Prédire
print(model.predict([[5]]))  # ≈ 15
